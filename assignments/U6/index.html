Ska jag använda Flexbox eller CSS Grid?

Flexbox och CSS Grid är två metoder för att bygga upp en layout med CSS. De jämförs ofta med varandra, trots att det är ganska stor skillnad mellan Flexbox och CSS Grid. I det här inlägget delar jag med mig av fördelar och nackdelar med de båda metoderna. Läs mer här.

För- & nackdelar med Flexbox

  • Bra för att strukturera innehåll (rubriker, stycken, länkar m.m.).
  • Endimensionella element: rad eller kolumn.
  • Inte bäst för tvådimensionella element.
  • Mindre kod än CSS Grid.
  • Flexbarn kan ha olika storlekar.

För- & nackdelar med CSS Grid

  • Bra för tvådimensionella element (rad + kolumn).
  • Kontroll över elementets "barn".
  • Kan vara förvirrande p.g.a. tvådimensionalitet (grid-row & grid-column).
  • Mindre bra för endimensionella element.

Slutsats

  • Båda bör användas! CSS Grid för större layouter, Flexbox för endimensionella layouter.
  • Grid ger kontroll över barn, Flexbox tillåter mer självständiga element med mindre kod.
