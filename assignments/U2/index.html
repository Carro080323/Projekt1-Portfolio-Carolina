Grundläggande element

Meta

html - huvudelement som innehåller hela html - filens element

head - innehåller metadata, länkar till CSS etc

body - innehåller allt som syns på webbsidan

style - inbäddad CSS

title - sidans titel

Layout

main - innehållet på sidan, innehåller flera sections

header - sidhuvud innehåller ev rubrik och ev information om sidan

section - sektion med innehåll

footer - beskriver sidfot

nav - navigerinsblock

ul och il - listor för navigering

span - används för rubrik i header

Text och Rubriker

p - paragraf

h1-h6 - Rubriker i olika storlekar

CSS

font-family – Typsnitt för text.

margin – Avstånd runt elementets kant.

margin-bottom – Avstånd under elementet.

padding – Avstånd mellan innehåll och kant.

background-color – Bakgrundsfärg.

border – Ramar runt elementet

border-radius – Rundade hörn.

display – Hur elementet visas

align-items – Vertikal justering i flexbox

justify-content – Horisontell justering i flexbox

list-style – Punktlista på/av (none tar bort punkter)

gap – Avstånd mellan flexbox-barn (t.ex. listpunkter)
